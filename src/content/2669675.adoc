== 문제
* 1부터 45까지의 숫자 중에 6개를 뽑는다.6개의 값이 다 달라야 한다.
* 출력시 작은 숫자부터 순서대로 출력

== 풀이
[source,python]
----
import random
balls = range(1,46)
random.shuffle(balls)
print sorted(balls[0:6])
----
